# Easy notion D-day widget
This repository provides very easy notion d-day widgets using only a single `.html` file. 


# How to use
## 1. Fork this repository
- Fork is much easier than clone for deploying it in Vercel

## 2. Deploy this index.html via Vercel (I think it enough for personal usage)
- Follow the steps in Vercel to connect your forked repository

## 3. Use the command \embed in notion
```
https://{your-address}.vercel.app/?title=Hello%20World!&date=2025-09-10
```
